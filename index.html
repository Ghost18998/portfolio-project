Project Overview

Goal: identify churn drivers and highlight segments at risk, using repeatable data prep and simple, trustworthy metrics.

  • Problem: churn rising; stakeholders lack visibility into why.
  • Approach: clean → segment → analyze → visualize → recommend.
  • Deliverables: KPI table, cohort/segment breakdowns, dashboard mock.
